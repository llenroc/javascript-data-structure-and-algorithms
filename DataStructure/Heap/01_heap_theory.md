### Heap Sort Algorithm

Array  = 3,5,1,2,7,8,4

            3
        5        1
      2   7    8   6


     index = 0              index =  1
            / \                     / \
           1   2                   3   4

     index =  i             
            /   \                     
          2i+1  2i+2                   
           


#### Steps
1. Heapify
2. Take first element in Sorted Array
3. Replace the first position with last element
4. Check if the tree is Heap ?
  If yes -> go to step 2
  If no  -> go to step 1.
